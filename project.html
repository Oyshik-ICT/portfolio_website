SimpleBanker

This is simple console-based bank application developed using OOP principles in python. Now I am learning OOP in python. So I created this using OOP. Through this project user can perform common banking operations like create new account, display all accounts, update an account, delete an account, deposit amount, withdraw amount. I have plans to expand and modify this project in the future as I learn more about oop concepts. Click the github icon to see the code.

ConsoleSchoolManager

This is console-based school management application developed using OOP principles in python. Through this a school can store the teachers, students and administrative staff (HR) information. Additionally, it provides functionalities to calculate total teacher salary, total HR salary, total student payment collection and total revenue. This project is currently under construction. Click the github icon to see the code.
